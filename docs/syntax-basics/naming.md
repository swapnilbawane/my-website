# Naming conventions

### Extensions of Components:
**type:** `.jsx`

**example:**
- `HomePage.jsx`
- `auth-context.jsx`
------
### Components:
**type:** PascalCase

**example:** 
- `HomePage.jsx` 
- `<HomePage />`
------
### CSS files:  
**type:** smallcase

**example:**  
- `index.css`
- `layout.css` 
------

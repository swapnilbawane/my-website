# Styled

## Buttons - styled

```javascript title="Buttons.jsx"

<button
className="button"
onClick={() => functionName(arguments)}>
  Add to Archive
</button>

<button
className="active-button"
onClick={() => functionName(arguments)}>
  Remove Archive
</button>

```

```javascript title="buttons.css"

.button {
color: white;
background-color: blue;

}

.active-button {
    color: white;
    background-color: black;
  }

```
